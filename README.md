# Pokemon Fetching App

This project is a React-based application that fetches Pok√©mon data from a public API. It demonstrates basic API integration and rendering of data in a simple interface.

## Prerequisites

Before you begin, ensure you have the following installed:

- [Node.js]
- [Bun] (for running and managing the project)

## Installation

1. Clone the repository
2. Navigate to the project directory
3. Install Dependencies
   (bun install)
4.Running the project
  (bun run dev)
